
Trading777 v5.2 更新日志
================================

一、版本号
---------
Trading777_v5.2.正式单机版.zip

二、本次修改内容
----------------

1. **首页 (index.html)**
   - 新增 **功能区3 · 个股情况**：
     * 在功能区2（目前持仓）与功能区4（交易记录）之间插入「个股情况」区块。
     * 动态扫描 `localStorage.trades` 中出现过的全部股票代码，按 **字母顺序** 生成按钮/标签，点击跳转到 `stock.html?symbol=xxx`。
   - 对应新增占位元素：`<div id="symbols-list" class="symbols-list"></div>`。

2. **样式表 (css/style.css)**
   - 新增 `.symbols-list` 及其子元素样式，支持横向排列及 flex‑wrap 换行，保持深色主题与绿色高亮一致。

3. **首页脚本 (js/dashboard.js)**
   - 新增 `renderSymbols()` 方法，负责渲染个股标签区。
   - `renderTrades()`：
     * 调整为 **日期倒序**（最新记录排在最前）。
     * 渲染完成后调用 `renderSymbols()`，确保列表实时更新。
   - 其余逻辑保持不变。

4. **交易记录页脚本 (js/trades.js)**
   - 新增日期倒序排序，保持与首页一致。
   - 在表格 **末尾**新增「详情」列，点击可跳转至 `stock.html` 个股详情页。

5. **HTML 其它文件**
   - 无变更。

三、未变动内容
--------------
- 功能区1（8 大 KPI 计算区）及功能区2（目前持仓）逻辑与样式保持原状。
- `stock.html` 个股详情页、数据导入/导出、价格自动刷新等既有功能未做任何修改。
- 文件结构与路径组织不变，可直接覆盖并上传 GitHub。

四、自查结果
------------
- **已本地跑通**，页面无 404 / 白屏。
- 各新增元素在深色背景下显示正常，窗口缩放/换行表现符合预期。
- 导入/编辑/删除交易后刷新页面，个股列表与交易排序均自动更新。
- 兼容此前 Key 与 API 逻辑，无额外依赖。
