
Trading777 v7.18 Update Log
==========================

Date: 2025-07-07

**New**
- **Historical Price Backfill** (`js/historicalBackfill.js`)
  * One‑off script to fetch *all* daily closes since **2025‑04‑07** for *every* tracked symbol.
  * Uses Finnhub `/stock/candle` (resolution D).  
  * Respects 50 req/min limit (`DELAY_PER_CALL_MS = 1200`).
  * Works in browser *or* CLI (`node js/historicalBackfill.js`).

- **priceService.js** enhancements
  * Added `fetchDailyCandles`, `saveDailyClosesBulk`, `getTrackedSymbols` helpers.

**Usage**
1. Ensure `KEY.txt` contains `FINNHUB=YOUR_API_KEY`.
2. Open site and run:
   ```js
   import { runHistoricalBackfill } from './js/historicalBackfill.js';
   runHistoricalBackfill(console.log);
   ```
   or `npm run backfill` if CLI integrated.

3. Script will throttle requests and write to `data/price-history.json`.

**Notes**
- Backfill script is idempotent. Existing dates will be overwritten with fresh data.
- No more Alpha Vantage for historical—Finnhub unified.
- Verified build & page integrity; regression‑tested homepage P/L calculation (module 6).
